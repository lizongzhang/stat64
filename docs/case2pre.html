1 数据和变量

数据来源：mpg {ggplot2}

样本容量：234

cty: city miles per gallon

hwy: highway miles per gallon

  • 英里每加仑，数值越大，代表越省油。

trans: type of transmission

drv: the type of drive train,

  • f = front-wheel drive,

  • r = rear wheel drive,

  • 4 = 4wd

2.1 cty均值的检验

gghistostats(mpg, cty, 
             test.value = 16,
             bin.args = list(color = "black",
                             fill = "cyan")) +
  theme_classic(base_size = 15)

2.1 cty均值的检验

\(H_0: \mu \leq 16\)

\(H_1: \mu > 16\)

拒绝原假设\(H_0: \mu \leq 16\) 。

在城市, 油耗均值大于16英里/加仑。

2.3 四驱和前驱城市道路油耗比较

  filter(drv %in% c("4", "f")) %>%
  ggbetweenstats(drv,

                 palette = "Set1",

2.3 四驱和前驱城市道路油耗比较

\(H_1: \mu_4 < \mu_f\)

在城市, \(\mu_4 < \mu_f\), 前驱(20)比四驱(14)省油。

5 驱动类型和气缸数量的关系分析

驱动类型和气缸数量不独立。

4缸，72%前驱，28%四驱;

6缸，54%前驱， 41%四驱，5%后驱;

8缸，1%前驱， 69%四驱，30%后驱。

6 研究结论

  • 油耗(单位：miles per gallon 英里每加仑), 数值越大代表越省油

  • 城市均值 > 16, 高速均值 > 22

  • 城市：四驱(14) < 前驱(20)

  • 高速：四驱(20) < 前驱(29)

  • 城市：自动挡(16) < 手动挡(19)

  • 高速：四驱(22) < 前驱(26 )

6 研究结论

  • 高速比城市道路省油

  • 前驱比四驱省油，高速(9)比城市(6)的差距更大。

  • 手动挡比自动档省油，高速(4)比城市(3)的差距更大。

6 研究结论

  • 四驱、前驱、后驱：45%，45%， 10%

  • 驱动类型和传动类型相互独立。

  • 驱动类型和气缸数量不独立。

    • 4缸，72%前驱，28%四驱

    • 6缸，54%前驱， 41%四驱，5%后驱

    • 8缸，1%前驱， 69%四驱，30%后驱
